Variavel

As variáveis no PHP são representadas por um cifrão ($) seguido pelo nome da variável.
Dica:Nome de variável não pode começar com caractere especial,número e underline ( _ ) .

Tipos de variaveis
        

          PHPINICIO

          $nome = "João";
          
          echo gettype($nome); // string
          
          $nome = 28;
          
          echo gettype($nome); // integer
          
          $nome = 78.500;
          
          echo gettype($nome); // float
          
          $nome = true;
          
          echo gettype($nome); /* boolean */
