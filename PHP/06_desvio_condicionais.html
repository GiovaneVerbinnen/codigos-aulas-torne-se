Desvio condicionais

O construtor If tem como objetivo executar todo o código que esteja entre o construtor caso o resultado da avaliação seja verdadeiro caso contrario nada será executado.

O else é a estrutura de controle que é executada quando o If, ou elseif,for retornado falso.

Exemplo de If simples e composta com Else 

      
        PHPINICIO
        $a = 5;
        if($a == 5){
            echo "O valor é 5";
        } 

        PHPFIM
        $a = 5;
        if($a == 5){  '#Se(condição)  True(verdade)'
            echo "O valor é 5";
        }else{
            echo "O valor é diferente";
        }

Operadores de comparação

  • == : Igualdade
  • > : Maior
  • < : Menor

  • <= : Menor Menor
  • != : Diferente

    • Exemplo

    Recebe a média de um aluno e verifica se o mesmo está aprovado ou reprovado
    Se quem a média for maior ou igual 7 escreva aluno aprovado , se for maior ou igual a 4 escreva aluno em recupreção caso contrario reprovado. 

            
            PHPINICIO
            $media =  3;
            if($media >= 7){
                echo "O aluno foi aprovado,com a média>". $media;
            }else if ($media >= 4) {
                echo "O aluno está de recupreração com média". $media;
            }else{
                echo "aluno reprovado";
            }
