If Ternario e Switch

O operador ternário basicamente é uma forma de escrever uma instrução if e else em uma única linha.

O switch funciona como uma série de if juntos, testando valores para uma variável ou expressão. No entanto o switch trabalha basicamente com o operador de igualdade. Então em casos que devemos testar se nossa variável, ou expressão, é igual a uma série de valores, o switch é uma boa saída.

Exemplo If Ternario

Se idade for maior ou igual a 18, liberado, senão censurado.

      
        PHPINICIO
        $idade = 18;
        $resultado = ($idade >= 18 ? "Liberado":"Censurado");
        echo $resultado;
        PHPFIM

Exemplo Switch

        
            PHPINICIO
        1    switch (expressão) {
        2        case valor1:
        3            //código a ser executado se a expressão for igual ao valor1
        4            break;
        5        case valor2:
        6            //código a ser executado se a expressão for igual ao valor2
        7            break;
        8        case valorN:
        9            //código a ser executado se a expressão for igual ao valorN
        10            break;
        11     }
            PHPFIM

Explicação

Linha 1: O parâmetro do switch/case deve estar entre parênteses e pode ser tanto uma variável quanto um método ou expressão;

Linhas 2, 5 e 8: A palavra reservada case seguida de um valor (numérico, string, etc) verifica se a expressão recebida como parâmetro é igual a esse valor. Se for, o trecho de código entre o case e o break logo abaixo será executado;

Linhas 3, 6 e 9: Os trechos de código entre o case e o break são executados caso a condição especificada no case seja atendida. Aqui pode haver várias linhas de código sem a necessidade de uso de chaves para delimitar o bloco;

Linhas 4, 7 e 10: O comando break faz com que a verificação seja encerrada caso a condição do case equivalente seja atendida. Isso faz com as demais condições não sejam avaliadas, pois como se trata de comparações de igualdade, apenas uma delas será atendida.
