Ambiente de Desenvolvimento

Para começar o desenvolvimento em C# é requisito ter como S.O instalado o Windows. Como IDE será utilizado o Visual Studio 2015 versão Community, caso a seu Windows seja inferior ao 8 o ambiente de desenvolvimento a ser instalado será o Visual Studio 2010.

Para download da IDE Visual Studio pode-se pesquisar pelo próprio site da Microsoft.

Tabs da IDE

Solution Explorer

É muito semelhante ao Explorer do Windows, para gerenciar seus projetos e arquivos.

Team Explorer

Em uma especie de “Git” da Microsoft, para gerenciamente de commit e deploys.

Class View

Mostra todas as classes que existentes no projeto

Propriedades

Mostra todas as propriedades de determinado objeto.

ToolBox

Mostra todos os tipos de objeto disponíveis para serem adicionados a página (em Windows form ou web form).

Server explore

Para a conecção com o Microsoft Azure.

Nuget package maneger

Está localizado na aba Tools, é um gerenciador de bibliotecas para o projeto.

Sitaxe
