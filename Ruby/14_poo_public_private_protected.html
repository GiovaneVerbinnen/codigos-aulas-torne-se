Modificadores de Acesso (public, private e protected)


Exemplos de acesso.

        
          class Carro
          protected
            def Danilo
              # método protegida
            end
          end

Usamos o sinal <, para representar o conceito de herança em Ruby

        
          class Fiesta < Carro
          end
          Fista.new.Carro # Não irá funcionar devido ao protected
